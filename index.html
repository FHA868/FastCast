FastCast - Документация

Getting Started

Эта документация поможет вам настроить и начать использовать FastCast.

Установка

Для использования, скачайте модуль из Roblox Library или с GitHub:

local FastCast = require(game.ReplicatedStorage.FastCast)

Пример использования


local hitbox = FastCast.new(workspace.Part)
hitbox:Fire()

API Reference

Методы

  • Fire() - Запускает рейкаст
  • Destroy() - Удаляет хитбокс
