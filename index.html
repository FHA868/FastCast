Usage

Template using FastCast for create hitbox:


local FastCast = require(game.ServerScriptService.FastCast)

local hitbox = FastCast.new(workspace.Part,false,"Test",{Distance=1000})--Third value can be "" or anything in string. In Table you need to type a Distance and Paramamenters for Raycast using RaycastParams.new()

hitbox.onHit:Connect(function(hit,distance)
    print(hit)--Printing a hit with a part or other objects in game
    print(distance)--Printing distance of raycast. Idk if it need so.
end)

hitbox:Fire()

API Reference

  • Fire(time) - Starts a function for raycast.Time can be nil!
  • Destroy() - Deletes all data for this hitbox.
  • onHit(function(hit,distance))-Gets object what hitted raycast.
  • MultiCast(directions,distance)- Makes multiple raycast in typed directions.

Returns Values

  • Fire(time) - Returns nothing use onHit to get object and distance.
  • Destroy() - Returns nothing. P.S.(Litteraly nothing to return)
  • onHit:Connect(function(hit,distance))-Returns a hitObject and distance of raycast.
  • MultiCast(directions,distance)- Returns all of results of raycast in table!
