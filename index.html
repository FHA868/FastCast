Getting Started

Здесь начинается ваша работа с FastCast. Чтобы установить модуль, используйте следующий код:

local FastCast = require(game.ReplicatedStorage.FastCast)

Установка

Скачайте модуль из Roblox Library или с GitHub.

local FastCast = require(game.ReplicatedStorage.FastCast)

Использование

Пример использования FastCast для создания hitbox:

local hitbox = FastCast.new(workspace.Part)

API Reference

  • Fire() - Запускает рейкаст.
  • Destroy() - Удаляет хитбокс.
